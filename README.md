# Fix-Oxygen-OS-Oneplus-Account

为 OxygenOS 16 修复中国大陆账号登录与本地化服务的 Magisk 模块

---

✨ 功能简介

Fix-Oxygen-OS-Oneplus-Account 是一个用于 修复 OxygenOS 16 账户中心无法登录中国大陆账号、恢复本地化服务 的 Magisk 模块。

由于 OxygenOS 国际版在系统层面移除了部分中国大陆相关组件，本模块通过补全必要文件与配置，使系统重新具备相关能力。

---

⚠️ 当前状态

> 模块仍在持续完善中，可能存在兼容性问题。

已知可能受影响的功能包括：

- 系统钱包部分功能  
- 游戏助手部分功能  
- 其他依赖国行框架的系统组件  

如需了解具体实现方式，请参考源码。

---

📦 版本说明

🔹 普通版（Stable）
- 仅修复账号登录与本地化服务  
- 不替换系统应用  
- 更适合追求稳定性的用户  

🔸 Alpha 版
- 在普通版基础上额外替换部分系统应用为国行版本  
- 包含但不限于：  
  - 软件商店  
  - 游戏助手  
- 更适合需要完整国行体验的用户  

> ⚠️ Alpha 版可能带来更多兼容性问题，请谨慎使用。
